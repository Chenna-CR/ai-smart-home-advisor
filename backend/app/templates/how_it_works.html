How It Works

How the Smart Home Assistant Works Today

The assistant turns a plain-language request into a smart home shortlist using AI intent analysis, live product search, strict smart-product filtering, and explainable ranking.

1
User Query
Type or speak a request like "best smart AC for living room"
2
AI NLP (Intent Extraction)
Groq extracts product type, features, budget, and use case
3
Search + Filter
Live shopping results are fetched and filtered to keep only relevant smart products
4
Ranking
Feature match, rating, price fit, reviews, and smart-home keywords are weighted
5
Explainable Results
Each recommendation gets a clear reason, pros, and cons
6
Saved History
Logged-in users and guests can revisit recent chats from the sidebar

What Changed in This Version

The app now includes guest mode, Google sign-in, recent chat history, compare support, and a tighter smart-product filter so the results stay focused.
